13. GEOS-Chem Diagnostics

GEOS-Chem v11-01 archives diagnostics to binary punch file format by default. We expect to retire the binary punch file format in the next version of GEOS-Chem, which will be called v11-02.

For the current list of diagnostic quantities that you can archive from GEOS-Chem simulations, please see the List of diagnostics for v11-01 on the GEOS-Chem wiki.

You can also archive emissions and other quantities computed by HEMCO directly to COARDS-compliant netCDF files. For more information, please see The HEMCO User's Guide: Diagnostics wiki page.
