Latest GEOS–Chem "Beta" Release

NOTE: Not all GEOS–Chem revisions are designated "public" releases. It is often expedient to stretch out updates to the code over several "beta" versions, so that each individual version may be independently benchmarked. You may download the latest GEOS–Chem beta version here; however, please be aware that each beta release comes "as-is".

Also note: please visit the Appendix 8 (Addenda) section of the GEOS–Chem Online Users' Guide. We typically only rewrite the GEOS–Chem Users' Guide whenever for new "public" releases. For the interim "beta" releases we add new information on the Addenda page.
