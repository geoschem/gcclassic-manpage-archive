4. Shared Data Directories

The GEOS-Chem shared data directories contain the various meteorological data fields, emissions data, and other data that GEOS-Chem will read in during the course of a simulation. Please see Chapter 2 for information about downloading GEOS-Chem shared data directories.

Please note that the shared data directory naming structure has changed in GEOS-Chem v10-01. Because we no longer need to store emission data on multiple grids, all shared data directories are now subdirectories of a single root directory called ExtData. Please see the Setting up the ExtData directory wiki page for detailed instructions on setting up your shared data directories so that they are compatible with GEOS-Chem v10-01.
