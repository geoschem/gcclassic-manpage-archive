8. GEOS-Chem Reference Guides

GEOS-Chem employs the ProTeX automatic documentation system to create highly detailed GEOS-Chem reference guides from comments in the source code documentation headers. You can build these reference guides in both PDF and PostScript formats by navigating to your source code directory (e.g. Code.v10-01) and typing:

make doc

You may also use the following links to access the GEOS-Chem v10-01 reference guides in PDF format::

  • Volume 1: GEOS-Chem Makefiles
  • Volume 2: GEOS-Chem Utility Modules
  • Volume 3: "Core" GEOS-Chem Modules and Routines
  • Volume 4: Global Terrestrial Mercury Model

For more information about the ProTeX system, please see the ProTeX automatic documentation system wiki page.
